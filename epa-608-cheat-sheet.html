üîß Type-Specific Key Points

Type 1 (Small Appliances)

  • Definition: ‚â§5 lbs refrigerant
  • Recovery: 0 psig OR 80%
  • Examples: Refrigerators, window AC, dehumidifiers
  • Disposal: Recover before disposal
  • Flammable refrigerants: R-600a, R-290 (caution)

Type 2 (High-Pressure)

  • R-22 recovery: 10" Hg vacuum
  • HFC recovery: 0 psig
  • Leak repair: 20% (comfort), 35% (commercial)
  • Evacuation: 500 microns (new install)
  • A2L refrigerants: R-32, R-454B

Type 3 (Low-Pressure Chillers)

  • Recovery requirement: 25 mm Hg absolute (0 psig)
  • R-123 classification: B1 (toxic), TLV = 50 ppm
  • Purge units: Remove air/non-condensables (systems operate below atmospheric pressure)
  • Leak test pressure: Maximum 10 psig (use nitrogen, NOT air)
  • Rupture discs: Pressure relief for chillers
